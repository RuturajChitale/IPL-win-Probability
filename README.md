# IPL Win Probability Predictor

🏏 A Machine Learning web application that predicts the live winning probability of IPL teams during the second innings.

## 🚀 Live Demo

🔗 **Try the app here:** (https://ipl-win-probability-kvkixneosy4dwetwythfcf.streamlit.app/)

## Features
- Live win probability prediction
- Interactive Streamlit interface
- Trained on historical IPL data
- Supports all current IPL teams

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
