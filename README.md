IPL Win Probability Predictor

This project is a Machine Learning-based web application that predicts the live winning probability of the batting and bowling teams during the second innings of an IPL match. The prediction is based on real-time match information such as the target score, current score, overs completed, wickets lost, venue, and the teams playing.

The model is trained on historical IPL match data and deployed using Streamlit to provide an interactive interface where users can simulate match scenarios and instantly view the winning probabilities of both teams.

Features
🏏 Live IPL win probability prediction
📊 Machine Learning model trained on historical IPL data
🎯 Predicts winning chances after every ball
🌍 Supports all current IPL teams
📍 Includes venue (city) information
💻 Interactive Streamlit web application
